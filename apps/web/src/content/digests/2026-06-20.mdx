    { label: "ui.sh", url: "https://ui.sh/" },

Tailwind Labs remplace le skill unique `/ui` par une série de skills installés localement, sans serveur MCP distant comme point de défaillance. `/design` reprend les règles couleur/espacement/typo/accessibilité ; suivent `/ideas`, `/brand-kit` (via gpt-image-2), `/componentize`, `/canonicalize-tailwind`, `/add-dark-mode`, `/make-responsive` et `/markup-from-image` (screenshot ou export Figma vers HTML/JSX propre). Le site ajoute aussi des comptes utilisateurs pour le partage de docs en équipe.
</NewsItem>

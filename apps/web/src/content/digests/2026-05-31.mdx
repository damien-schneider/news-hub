<NewsItem
  title="Pour les agents de code, le terminal remplace le RAG"
  category="ia"
  tags={["agents", "rag", "papers"]}
  sources={[{ label: "arXiv (GrepSeek)", url: "https://arxiv.org/abs/2605.05242" }, { label: "via AlphaSignal", url: "https://alphasignal.ai" }]}
>
Le RAG classique découpe les documents en chunks et filtre, avant la boucle de raisonnement, les données syntaxiques dont un agent a besoin : code d'erreur exact, chemin de fichier, contrainte de version. Le **Direct Corpus Interaction** contourne les embeddings et donne à l'agent un accès direct au corpus via `grep`, `find`, `cat`, `sed`. Le framework [GrepSeek](https://arxiv.org/abs/2605.05242) entraîne le modèle à émettre des commandes shell, et son moteur sharded-parallèle accélère la recherche jusqu'à **7,6×**. Approche recommandée à grande échelle : retrieval sémantique pour l'ancrage, DCI pour la vérification de précision.
</NewsItem>

Elle reproduit une variante génétique naturelle qui protège à vie du risque coronarien. Phase 2 prévue d'ici fin 2026.

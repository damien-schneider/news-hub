  title="Anthropic redéploie Claude Fable 5 dans le monde avec un nouveau garde-fou"
  category="ia"
  tags={["anthropic", "sécurité"]}
  sources={[{ label: "Anthropic", url: "https://www.anthropic.com/news/redeploying-fable-5" }, { label: "via AlphaSignal", url: "https://alphasignal.ai" }]}

  imageAlt="Claude Fable 5"
>
Claude Fable 5 est de nouveau en ligne après **19 jours** de suspension. Des chercheurs d'Amazon avaient contourné ses filtres de sûreté pour lui faire identifier des failles logicielles et, dans un cas, produire du code d'exploitation ; le gouvernement américain avait traité l'affaire comme un risque de sécurité nationale et retiré le modèle partout. Pour le remettre en service, Anthropic ajoute un classifieur de sûreté entraîné sur ce contournement (bloqué dans **plus de 99 %** des cas), fait temporairement retomber les requêtes de code sensibles sur **Opus 4.8**, et coécrit avec Amazon, Microsoft et Google un cadre commun de gravité des jailbreaks. Le modèle est inclus dans les offres de base **jusqu'au 7 juillet**, puis facturé à l'usage.
</NewsItem>

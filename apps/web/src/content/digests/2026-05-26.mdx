  category="ia"
  tags={["xai", "agent", "cli"]}
  sources={[{ label: "xAI", url: "https://x.ai/news/grok-build-cli" }, { label: "via AlphaSignal", url: "https://alphasignal.ai" }]}
>
xAI ouvre la bêta de [Grok Build](https://x.ai/cli), un agent de code en terminal qui lit le projet, planifie, édite les fichiers et lance des commandes. Son **Plan Mode** inverse le schéma habituel : l'agent expose tout le plan, l'utilisateur approuve ou réécrit chaque étape, et rien n'est touché avant validation. Sous-agents parallèles, mode headless (`-p`) pour la CI, support des MCP et lecture native des fichiers `CLAUDE.md` existants. Réservé aux abonnés SuperGrok, sur `grok-build-0.1` (contexte 256K).
